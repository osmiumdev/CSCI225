Description:

This program is an interactive calendar which allows users to create and RSVP for events.

Features include: Creating events, modifying existing events, deleting your own events,

leaving comments, RSVPing for events, changing the color of events, and more!

Additionally, admins may override and modify other's events in the case of inappropriate content.
